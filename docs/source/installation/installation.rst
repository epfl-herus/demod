
.. _installation:
===============
Installation
===============

Here we explain the installation.


Requirements
------------

You will need to have
`Python 3 installed <https://wiki.python.org/moin/BeginnersGuide>`_

Demod supports python versions >=3.8.

Install with pip
----------------

You can install our library with pip.

.. code-block::

    pip install demod
