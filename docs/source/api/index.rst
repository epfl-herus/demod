=============
API Reference
=============

The API reference provides detailed descriptions of classes and
functions. It should be helpful if you plan to create your own
simulations or add your custom dataset.

Simulators shows the simulators you can use to create scenarios.
Abstract simulators help you creating you own simulation algorithms.
Datasets show you which data is available.
Utils contains different utility classes and functions usefull for
developpement.
